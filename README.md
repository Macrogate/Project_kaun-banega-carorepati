This project is a text-based C++ implementation of the world-famous quiz show **"Kaun Banega Crorepati."** The game simulates the experience of sitting in the "Hot Seat," where players must answer a series of 20 increasingly difficult questions to win the grand prize of **10 Crore Rupees**.

The program uses structured conditional logic to track progress and determine the final winning amount based on the player's performance.
